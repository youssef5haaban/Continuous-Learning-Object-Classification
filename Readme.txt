## Running the Experiment

To run the experiment, follow these steps:

1. Open the `test-slda.ipynb` file in a Jupyter Notebook environment.

2. Execute the cells in the notebook sequentially to perform the experiment.

## Web Deployment

To deploy the project as a web application, follow these steps:
1. Make sure you have Streamlit installed. If not, you can install it using:
`pip install streamlit`
2. Run the vision3.py script using Streamlit:
`streamlit run vision3.py`
